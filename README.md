# POSITION

# Overview

Finds what index a given number would go into for the given sorted array.

# Installation

Include the following script in your html file:
```
<script src="position.js"></script>
```

# Usage

Example calls:
```
var list = [1,3,6,8]
console.log(position(list, 5)) // returns 2
```
